# **바코드 생성기와 스캐너**


zxing 라이브러리를 이용한 바코드 생성기와 스캐너입니다.

구현한 방법은 기존에 여느 바코드 개발과 동일합니다. 다만, 스캐너에서 스캔화면을 유지하면 스캔을 하고, 결과를 받으며, 임의의 시간(1초) 후에 다시 스캔 상태가 되어 다음 스캔을 이어 계속하도록 구현하였습니다.
